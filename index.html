SlappyCow

A rip-off version of a Vietnamese game - Flappy Bird, by Dong Nguyen (Nguyễn Hà Đông).

Umm, but also, this is the place where cows can fly UFOs, control them, drive through the obstacles and conquer the mother of cosmos.

THE SLAPPYCOW HOMEPAGE

Scroll down for the download guide, how-to-play, credits and patch notes!

How did we come up with this stupid idea?

As I have said, that sounds kinda fucking impossible. But for us, MKDC, we can easily nail down the game with some simple finger snaps.

With many epic features, simple and comfortable GUI, an idea about a game named "SlappyCow" has been come up with.

So, let’s get started.

What differences have we done so far?

This game was inspired by the OG one - Flappy Bird. We have upgraded the assets, been more creative in designing textures!

Btw, our character is special af. A cow which is able to fly an Unidentified Flying Object :)?

Download

There are several ways to download this game.

The most familiar way is from Google Drive (ye every1 should know that).

A more complicated way, but instead we could have the most stable version of the game is from GitHub.

How to play?

Simply just do left-click. An invisible force will help our cow fly up. (don't ask why)

Also there're some configurable properties, but I would love to explain those later =)
